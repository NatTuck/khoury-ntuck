Bottlenose

Bottlenose is an online submission management tool for university computer science courses. It allows for web submission and automatic grading of programming projects, as well as instructor grading and feedback.

Bottlenose has been in use in the CS department at UMass Lowell for various courses for the past several years.

source @ github

Pancake

Pancake is a library that performs just in time specialization of OpenCL kernels. This can provide significant speedups on some algorithms.

source @ github

FogSync

FogSync is a cloud based file synchronizaton tool designed to be secure against the provider. It's currently in early development, but it should be really cool once it's ready.
